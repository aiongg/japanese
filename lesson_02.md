# Lesson 2: Navigating Train Stations and Airports

こんにちは, エドワード! Welcome to Lesson 2. Today, we’ll build on your skills by focusing on practical Japanese for train stations (*えき*) and airports (*くうこう*). These are key scenarios for your vacation, covering buying tickets, asking about platforms and facilities, validating passes, checking in, and handling immigration/customs. Since you’re already comfortable with Hiragana and Katakana, and have some Kanji recognition from Mandarin, we’ll use this to help you navigate Japan with your family. Let’s dive in with dialogues and examples to get you comfortable with the language!

### 文法 (*ぶんぽう*) のポイント (*ポイント*) - Grammar Points

1. **～まで - Indicating Destination or Extent:**  
The particle ～まで marks a destination or a limit, such as a place you’re going "to" or an extent of time/space. It’s often used with transportation or actions.

- 東京まで切符をください。  
*(Please give me a ticket to Tokyo.)*

| Vocabulary | Kana       | English Gloss |
|------------|------------|---------------|
| 東京       | とうきょう | Tokyo         |
| まで       | -          | until/to      |
| 切符       | きっぷ     | ticket        |
| を         | -          | object marker |
| ください   | -          | please (give me) |

- 大阪まで行きます。  
*(I’ll go to Osaka.)*

| Vocabulary | Kana       | English Gloss |
|------------|------------|---------------|
| 大阪       | おおさか   | Osaka         |
| まで       | -          | until/to      |
| 行きます   | いきます   | go            |

- 駅まで歩きます。  
*(I’ll walk to the station.)*

| Vocabulary | Kana       | English Gloss |
|------------|------------|---------------|
| 駅         | えき       | station       |
| まで       | -          | until/to      |
| 歩きます   | あるきます | walk          |

2. **～たいです - Expressing Desire:**  
The ～たいです form expresses a personal desire to do something. It’s made by adding ～たい to a verb stem and ending with です for politeness, useful for stating what you want during travel.

- 荷物を預けたいです。  
*(I want to store my luggage.)*

| Vocabulary | Kana       | English Gloss |
|------------|------------|---------------|
| 荷物       | にもつ     | luggage       |
| を         | -          | object marker |
| 預けたい   | あずけたい | want to store |
| です       | -          | is (copula)   |

- 日本に行きたいです。  
*(I want to go to Japan.)*

| Vocabulary | Kana       | English Gloss |
|------------|------------|---------------|
| 日本       | にほん     | Japan         |
| に         | -          | particle (destination) |
| 行きたい   | いきたい   | want to go    |
| です       | -          | is (copula)   |

- 水を飲みたいです。  
*(I want to drink water.)*

| Vocabulary | Kana       | English Gloss |
|------------|------------|---------------|
| 水         | みず       | water         |
| を         | -          | object marker |
| 飲みたい   | のみたい   | want to drink |
| です       | -          | is (copula)   |

3. **～できますか - Asking About Ability or Permission:**  
The ～できますか structure asks if something is possible or allowed. It’s formed by adding ～できる (can do) to a verb or noun and ending with か, perfect for checking options at stations or airports.

- 切符は新幹線で使えますか？  
*(Can this ticket be used for the Shinkansen?)*

| Vocabulary | Kana       | English Gloss |
|------------|------------|---------------|
| 切符       | きっぷ     | ticket        |
| は         | -          | topic marker  |
| 新幹線     | しんかんせん | Shinkansen    |
| で         | -          | by means of (particle) |
| 使えます   | つかえます | can use       |
| か         | -          | question marker |

- ここで写真を撮れますか？
